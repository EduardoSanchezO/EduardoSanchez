Research Assistant

Cinvestav IPN / Intel Labs 7/2021-Present

Humanoid autonomous navigation involves pathfinding in the robot's state space, with the goal of avoiding obstacles and ensuring feasible trajectories. One efficient search technique is Rapidly Exploring Random Trees (RRT), which connects obstacle-free and stable nodes in the state space towards a pre-established goal. However, checking node stability and collision-free conditions can be computationally challenging due to the random nature of RRT. To address this, Reinforcement Learning is implemented, where a neural network learns which nodes to provide to RRT, significantly accelerating the trajectory search process.

Technologies Applied:

Python 3.7,Open cv 2,Pytorch,Proximal Policy Optimization,Ros 20.04.

Detection and identification of obstacles are essential components in humanoid robot navigation, which is why an obstacle detection system using neural networks was implemented. These obstacles are identified and mapped onto an image, which is subsequently used for trajectory planning. The neural network architecture used for this task was Yolo.v7, trained to identify 10 different objects as obstacles.
